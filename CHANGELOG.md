# Changelog

All notable changes to ocview will be documented in this file.

Format based on [Keep a Changelog](https://keepachangelog.com/).

## [0.1.0] - 2025-01-01

### Added
- GTK3 + WebKitGTK 4.1 WebView window with auto right-half positioning
- FastAPI bridge with 15 REST endpoints for remote control
- 13 MCP tools for full OpenCode AI integration
- File watcher with debounced auto reload (watchdog)
- Window state persistence across restarts
- Console error capture and AI forwarding
- Smart `describe_page()` DOM summary for AI
- Screenshot capture via MCP (full page PNG)
- One-command installer for 6 Linux distros
- `--update`, `--version`, `--help` CLI commands
- Clean uninstaller with config keep option
- GitHub issue templates, PR template, CI workflow
